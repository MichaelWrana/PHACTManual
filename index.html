User Access
Gain access to the PHACT Web portal using your account. View personal account information and access administrator features. Signing off and switching accounts.
Dashboard
The Homepage of the PHACT Portal. View a heatmap of the target environment. See detailed information about each installed node. See environment capacity gauges.
History
Overview of historical recorded information from each node in the target environment. See graphs representing number of contacts near each node.
Incidents
See all incidents that have occured in the target enviornment. Respond to an incident from a computer terminal and assign incidents to users.
Contact Tracing
Enter a device in the target environment and a timeframe to view a history of the other devices they have interacted with. Notify relevant parties of potential infection.
Manage Users
Create new user accounts and remove existing ones. Control user permissions and website access. View a user's incident reporting history.
Node Settings
View and customize the settings associated with each node in the environment. Customize the breakpoints for creating an incident. Add detailed notes for each node.
PHACT Mobile App
Downloading and accessing the associated PHACT mobile app. Remotely updating the status of an incident and requesting assistance.
FAQs
The most important features needed by users quickly summarized. If you dont know how to do something, the answer is probably here!
